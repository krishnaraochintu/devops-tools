Dev Tools Cheatsheet

Bash
  • List files: ls -la
  • Find text: grep "text" file
  • Change dir: cd /path
  • Copy file: cp src dest
  • Move file: mv src dest
  • Remove file: rm file
  • Show disk usage: du -sh *
  • Show free space: df -h
  • Find files: find . -name "*.txt"
  • Show running processes: ps aux
  • Kill process: kill -9 <pid>
  • Archive files: tar -czvf out.tar.gz dir/
Ubuntu
  • Update packages: sudo apt update
  • Upgrade packages: sudo apt upgrade
  • Install package: sudo apt install <pkg>
  • Remove package: sudo apt remove <pkg>
  • Show disk usage: df -h
  • Show memory usage: free -h
  • Reboot: sudo reboot
  • Shutdown: sudo shutdown now
  • Check system info: uname -a
  • List users: cat /etc/passwd
Vim
  • Insert mode: i
  • Save & quit: :wq
  • Quit without saving: :q!
  • Search: /text
  • Replace: :%s/old/new/g
  • Undo: u
  • Redo: Ctrl+r
  • Delete line: dd
  • Copy line: yy
  • Paste line: p
Git
  • Clone repo: git clone <url>
  • Status: git status
  • Commit: git commit -m "msg"
  • Push: git push
  • Pull: git pull
  • Log: git log --oneline
  • Branch: git branch
  • Checkout: git checkout <branch>
  • Merge: git merge <branch>
  • Stash: git stash
  • Show remote: git remote -v
  • Show diff: git diff
Docker
  • Build image: docker build -t name .
  • Run container: docker run -it name
  • List containers: docker ps -a
  • Remove image: docker rmi name
  • Stop container: docker stop <container>
  • Remove container: docker rm <container>
  • Exec into container: docker exec -it <container> bash
  • View logs: docker logs <container>
  • List images: docker images
  • Prune unused: docker system prune
Kubernetes
  • Get pods: kubectl get pods
  • Get services: kubectl get svc
  • Describe pod: kubectl describe pod <name>
  • Apply manifest: kubectl apply -f file.yaml
  • Delete pod: kubectl delete pod <name>
  • Logs: kubectl logs <pod>
  • Exec into pod: kubectl exec -it <pod> -- bash
  • Get nodes: kubectl get nodes
  • Scale deployment: kubectl scale deployment <name> --replicas=3
  • Port forward: kubectl port-forward svc/<name> 8080:80
Ansible
  • Run playbook: ansible-playbook playbook.yml
  • Ping hosts: ansible all -m ping
  • List hosts: ansible-inventory --list
  • Ad-hoc command: ansible all -m shell -a 'uptime'
  • Check mode: ansible-playbook playbook.yml --check
Terraform
  • Init: terraform init
  • Plan: terraform plan
  • Apply: terraform apply
  • Destroy: terraform destroy
  • Show state: terraform show
  • List resources: terraform state list
Helm
  • List releases: helm list
  • Install chart: helm install name chart
  • Upgrade release: helm upgrade name chart
  • Uninstall release: helm uninstall name
  • Show values: helm show values chart
Jenkins
  • Restart Jenkins: sudo systemctl restart jenkins
  • View logs: sudo journalctl -u jenkins
  • Install plugin: Manage Jenkins > Manage Plugins
  • Reload config: Manage Jenkins > Reload Configuration from Disk
AWS CLI
  • Configure: aws configure
  • List S3 buckets: aws s3 ls
  • Copy to S3: aws s3 cp file.txt s3://bucket/
  • List EC2: aws ec2 describe-instances
  • List IAM users: aws iam list-users
